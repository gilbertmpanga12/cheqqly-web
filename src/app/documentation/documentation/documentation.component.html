#1 Get Cheqqly snippet from sidebar menu on dashboard and configure

Cheqqly snippet in configuration requires :

  1. Amount   number*
  2. MerchantId   string*
  3. Business name   string*
var config = {

merchantId: 'YOUR-MERCHANT-ID', 

amount: 'AMOUNT',

businessName: 'YOUR-BUSINESS-NAME'

}

Use 4242 4242 4242 4242 as a test credit card. This is for testing the sdk if everything is set up well You can test here*

Get your merchantId from settings

Every amount is 1/100 of Amount. e.g $5 would be configured as 500. 1/100 * 500 => would yield $5. Please note that the least amount accepted is $0.50 *

{ message:`Successully paid _amount_ to business_name` }

{ message:`Failed to process payment. Please check if you entered right card details or check is card is credited` }
